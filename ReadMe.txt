root
    根目錄,這裡放頁面的html跟php,沒有其他東西了
    頁面的html跟php也只能放這裡,不許放其他資料夾
    L Ai_Ps
        不許放個別檔案,只能放資料夾
        L   用頁面名稱或是功能名稱建立資料夾,
            .ai & .psd 或是其他繪圖檔案原始檔可以放這,
            但請只放極度重要的，不重要的不用同步上來
    L css
        scss會自動產出,這個資料夾不用動
        除非有引入別的css進來
        引入進來的也不用特別分類
        檔名標示清楚在哪個頁面底下要用就行
    L fonts
        字型丟這,直接丟進來
    L images
        不許放個別檔案,只能放資料夾
        L   用頁面名稱或是功能名稱建立資料夾, png,jpg,ttf,gif ...
            啥圖都可以丟,就是頁面會讀到要用的圖就放資料夾裡
    L INDIVIDUAL
        這個資料夾是給個人存不可被動,
        但又需要共享的檔案,
        這裡的資料夾請以個人名字做分類
        請除了建立者本人以外的人不要碰
        (像是建泓會上傳建立資料庫的一些檔案)
    L js
        檔案名稱: 頁面.js 或 部件.js
    L json
        只能放.json檔
    L palette
        放色票
    L sass
        頁面.scss 或是 部件.scss
        .sass也可以啦 都放這
